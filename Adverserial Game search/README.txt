Tic-Tac-Toe against adverserial agents is a part of the AI parcticum of IIT palakkad


Tic.py : Contains the Interface for two people to play Tic-Tac-Toe
Minimax.py: Contains the minimax adversarial agent code
AlphaBeta.py: Contains the alpha beta pruned adversarial agent code
Game_against_minimax_adversary.py: File to play Tic-Tac-Toe against a minimax adverserial agent
Game_against_alpha_beta_pruned_adversary.py: File to play Tic-Tac-Toe against a Alpha Beta pruned adverserial agent

In each game winner gets 100 pts loser gets -100 pts in case of draw both players get 50
